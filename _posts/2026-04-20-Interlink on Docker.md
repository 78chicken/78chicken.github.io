**Interlink Labs** 是一個主打 **去中心化身份（Decentralized Identity）與 AI 訓練** 的平台。  
與傳統掛機不同，它不只要求你的頻寬或算力，更強調「**真實人類參與**」。透過生物識別技術（人臉識別），它確保網路中的每個節點都是真實的人，而非機器人。

這類模式屬於 **DePIN（Decentralized Physical Infrastructure Network）** 的進階應用，特色包括：

* **人類算力（Human Hash Power, HHP）**：以人類身份作為挖礦權益。
* **聯合學習（Federated Learning）**：在不洩露隱私的情況下，利用分散的手機端數據訓練 AI 模型。
* **身份基礎設施**：建立一個名為 InterLink ID 的通用數位身份。

簡單說，Interlink 想做的是「AI 時代的真實人類身分證」，並讓參與驗證的用戶分享 AI 成長的收益。  
目前專案正處於 **早期挖礦階段（Airdrop Phase）**，預計在 2026 年 Q2 進行 TGE，建議盡早進場佈局。

## 🧠 Interlink 是怎麼運作的？（How It Works）

根據官方說明，Interlink 的基本流程如下：

1.  **建立 InterLink ID**：下載 App 並設定專屬 ID（4-12 位字元）。
2.  **生物識別驗證**：通過 3D 人臉掃描，證明你是真實的人類，啟動 **HHP（Human Hash Power）**。
3.  **週期性挖礦**：每 24 小時點擊一次挖礦，系統會定期要求重新驗證人類身份。
4.  **貢獻數據/模型訓練**：手機會執行簡單的 AI 訓練任務（如標註或本地模型微調）。
5.  **領取 $ITLG**：根據你的 HHP 值與活躍度分配獎勵。

官方宣稱：

* **零成本參與**：不需要昂貴的 GPU，手機即可挖礦。
* **抗機器人機制**：嚴格的 ZK-Proof（零知識證明）生物驗證，防止工作室批量刷號。
* **隱私保護**：生物特徵資料會被轉換為加密 Hash，不儲存原始人臉影像。

### 💰 代幣經濟學

Interlink 採用 **雙代幣模型**：

* **$ITLG (Genesis Token)**：主要獎勵代幣，用於初期挖礦與社群激勵。
* **$ITL (Official Token)**：未來的官方治理與質押代幣，預計 2026 Q2 上線。

目前挖礦資訊：
* **基礎收益**：每節（Epoch）可獲得固定 $ITLG。
* **HHP 加成**：推薦朋友或參與特定任務可提升 HHP，進而增加挖礦產量。
* **TGE 時間**：預計 2026 年第二季。

---

### 🤝 推薦與團隊獎勵

Interlink 提供兩層推薦機制與組隊加成：

* **直接推薦 (L1)**：獲得額外 $ITLG 獎勵與 HHP 加成。
* **群組挖礦**：組成「Security Group」或加入社群團隊可獲得更高的產量提升（Boost）。

這也是目前社群推廣的主要方式，人脈越廣，HHP 越高。

## ⚠️ Interlink 的缺點與風險

1.  **隱私擔憂**：雖然官方宣稱使用 ZK-Proof，但仍需掃描人臉，對隱私極度敏感者需自行評估。
2.  **頻繁操作**：目前需要每 24 小時手動點擊一次，不像完全自動化的 Docker 掛機軟體。
3.  **代幣價值未知**：在正式 TGE 並上架交易所前，所有 $ITLG 僅為點數性質，存在空投不如預期的風險。
4.  **需刷臉** 無法大量佈署,機掰雞只用一台,若要一起建立Group來增加挖礦效率,請在留言區留下你的ID

## 🐳 Docker 執行Setup指令

```bash
# -v /opt/interlink/accounts.json 請換成你自己的路徑
docker run -d -m 50M --pull=always \
--name Interlink \
-e Mode=setup
-v /opt/interlink/accounts.json:/app/interlink/accounts.json \
docker.io/78chicken/interlink:latest
```
